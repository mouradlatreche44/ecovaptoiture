# Ecovaptoiture — Site vitrine

Site vitrine pour **Ecovaptoiture**, spécialiste du nettoyage écologique de toitures à la vapeur basse pression dans les Landes (40) et les Pyrénées-Atlantiques (64).

## Stack

- HTML5 statique (7 pages)
- Tailwind CSS via CDN (palette `brand` vert + `accent` bleu, matchant le logo)
- Lucide Icons via CDN
- Google Fonts : Baloo 2 + Inter

## Pages

1. `index.html` — Accueil
2. `nettoyage-toiture.html` — Nettoyage vapeur (toitures, terrasses, bardages, démoussage)
3. `gouttieres.html` — Entretien de gouttières
4. `hydrofuge.html` — Hydrofuge toiture & façade + anti-mousse
5. `a-propos.html` — À propos
6. `contact.html` — Contact + devis
7. `mentions-legales.html` — Mentions légales
